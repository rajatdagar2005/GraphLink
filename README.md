# GraphLink Pro

An interactive Graph Visualisation Tool built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add nodes and edges
- Drag and visualize graph
- BFS & DFS animations
- Shortest path finder

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
